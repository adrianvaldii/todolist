# Todolist Application

Simply Todolist Apps using Laravel 5.2 and Ajax

## How to Use

+ Config your database (File .env)
example : 
- DB_HOST=127.0.0.1
- DB_PORT=3310
- DB_DATABASE=todolist
- DB_USERNAME=root
- DB_PASSWORD=Valdi@99

+ Using the Apps

1. Run the Apps using "php artisan serve"
